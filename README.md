# Modulación en Amplitud (AM) en MATLAB

Este proyecto implementa un sistema de modulación en amplitud (AM),
incluyendo el análisis de la señal en el dominio del tiempo y la frecuencia.

## Contenido
- Generación de señal de mensaje
- Modulación AM
- Análisis espectral
- Introducción de ruido
- Atenuación y distorsión

## Requisitos
- MATLAB
- Signal Processing Toolbox
